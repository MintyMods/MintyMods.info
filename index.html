User Defined Remote Views

Create custom views showing your selected sensor information via Gauges, Charts, Tables or just plain labels.

Remote view support via Browser, Mobile, or T.V.

User Defined Notifications / Alerts

Create custom alerts for sensors that deviate from predetermined minimum and/or maximum ranges. Different notification levels can be configured for info , warning, & critical e.g. desktop push notifications

Cloud triggers enable actions via IFTTT or Amazon Web Services

Turn your lights red when your CPU overheats

Sensor History Recording

Sensor results are stored in a local database allowing full queries to be performed against previous recordingsSs

Polling allows consistent history recording even while not using the views

Schedule recording of sensors every 30 seconds

API / SDK

RESTful service allows realtime remote reporting of current sensor values or queries against any previous sensor history
