User Defined Views

Create custom remote views showing your selected sensor information via Gauges, Charts, Tables or just plain labels. Views can target specific display sizes to allow remote monitoring via PC, Mobile, or T.V.

User Defined Alerts

Create custom notifications for sensors that deviate from your minimum or maximum predetermined ranges

Cloud Triggers for IFTTT actions - Turn your lights red when your CPU overheats

Sensor History Recording

Sensor results are stored in a local database allowing full queries to be performed against any previous recordings

Scheduling allows consistent recording of sensor information even while not actively using the views. Schedule sensor recording at pre-determined intervals

Secure

Authenticated User Access Control - HTTPS over TLS

RESTfull API

RESTfull Web Service allows remote reporting of realtime sensor values or queries against sensor history - Supports Authentication
